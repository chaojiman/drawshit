✅ 1. 绘画功能 (index.html)
   • 16种颜色选择
   • 画笔粗细调节 (1-30px)
   • 橡皮擦工具
   • 撤销功能 (最多20步)
   • 清空画布
   • 触摸屏支持
   • 作品署名和保存

✅ 2. 幕布动画 (tank.html) ⭐核心亮点⭐
   • 【每秒产出4个大便】(250ms间隔)
   • 8种不同风格的搞笑大便
   • 真实物理掉落效果
   • 重力加速和旋转动画
   • 落地检测和淡出效果
   • 开始/停止控制
   • 键盘快捷键支持

✅ 3. 排行榜系统 (rank.html)
   • 投票系统 (点赞/点踩)
   • 防重复投票
   • 按票数/时间排序
   • 前三名特殊标记
   • 自动刷新功能

✅ 4. 8种大便样式 ⭐让网站充满活力⭐
   1. 经典大便 😊 - 棕色三层堆叠，带笑脸
   2. 螺旋大便 🌀 - 螺旋形状，金色闪光
   3. 愤怒大便 😠 - 红色，冒烟，皱眉
   4. 快乐大便 😄 - 金色渐变，大笑
   5. 高级大便 🎩 - 礼帽和单片眼镜
   6. 像素大便 👾 - 8-bit复古风格
   7. 彩虹大便 🌈 - 七彩渐变，星星闪光
   8. 怪物大便 👹 - 多眼睛和触手

【物理动画系统】(src/js/tank.js)
┌────────────────────────────────────────────────────────────────┐
│ class FallingPoop {                                            │
│     update() {                                                 │
│         if (this.hasLanded) {                                  │
│             // 落地后淡出效果                                  │
│             this.opacity -= 0.005;                             │
│             this.y += 0.5;                                     │
│         } else {                                               │
│             // 掉落中                                          │
│             this.y += this.velocityY;                          │
│             this.rotation += this.rotationSpeed;               │
│             this.velocityY += 0.1;  // 重力加速                │
│                                                                │
│             // 落地检测                                        │
│             if (this.y >= containerHeight - ...) {             │
│                 this.hasLanded = true;                         │
│             }                                                  │
│         }                                                      │
│         this.updatePosition();                                 │
│     }                                                          │
│ }                                                              │
└────────────────────────────────────────────────────────────────┘
