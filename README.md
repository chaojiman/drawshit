# DrawShit 💩

一个有趣的在线绘画网站，灵感来源于 drawafish.com，但是我们画的是大便！

## 功能特性 ✨

### 🎨 画大便页面 (index.html)
- 完整的绘画工具：多种颜色、橡皮擦、撤销、清空
- 可调节画笔粗细
- 支持鼠标和触摸屏绘画
- 提交作品并署名

### 🎭 幕布页面 (tank.html)
- **每秒产出4个不同样式的搞笑大便**
- 大便从天空掉落的动画效果
- 8种不同风格的大便：
  - 经典大便 😊
  - 螺旋大便 🌀
  - 愤怒大便 😠
  - 快乐大便 😄
  - 高级大便 🎩
  - 像素大便 👾
  - 彩虹大便 🌈
  - 怪物大便 👹
- 实时物理效果和旋转动画
- 支持开始/停止控制
- 键盘快捷键：空格键开始/停止，C键清空

### 🏆 排行榜页面 (rank.html)
- 投票系统（点赞/点踩）
- 按票数或时间排序
- 显示创作者和创作时间
- 前三名特殊标记（金银铜牌）
- 自动刷新

## 技术实现 🔧

- **纯前端实现**：HTML5 + CSS3 + Vanilla JavaScript
- **Canvas API**：用于绘画和渲染
- **LocalStorage**：存储作品和投票数据
- **动画**：requestAnimationFrame 实现流畅动画
- **响应式设计**：支持桌面和移动设备

## 如何使用 📖

1. **画大便**：
   - 打开 `index.html`
   - 选择颜色和画笔粗细
   - 在画布上画出你的大便
   - 点击"扔下去！"提交作品

2. **看大便掉落**：
   - 打开 `tank.html`
   - 点击"开始下大便雨！"
   - 观看大便从天空掉落
   - 每秒会产生4个不同样式的大便

3. **投票排行**：
   - 打开 `rank.html`
   - 浏览所有作品
   - 为你喜欢的大便投票
   - 查看排行榜

## 本地运行 🚀

直接用浏览器打开 `index.html` 即可，无需服务器！

或者使用简单的 HTTP 服务器：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (如果安装了 http-server)
npx http-server

# 然后访问 http://localhost:8000
```

## 特色功能 🌟

### 每秒4个大便
在幕布页面，我们实现了**每秒产出4个不同样式的搞笑大便**：
- 使用 `setInterval(250ms)` 实现每250毫秒生成一个大便
- 8种随机生成的大便风格，让画面充满活力
- 真实的物理掉落效果和旋转动画
- 落地后会慢慢淡出和下沉

### 搞笑大便样式
- **经典大便**：棕色三层堆叠，带笑脸
- **螺旋大便**：螺旋形状，带金色闪光
- **愤怒大便**：红色，冒烟，皱眉
- **快乐大便**：金色渐变，大笑，腮红
- **高级大便**：戴礼帽和单片眼镜，有小胡子
- **像素大便**：8-bit复古风格
- **彩虹大便**：七彩渐变，带星星闪光
- **怪物大便**：紫色，多只眼睛和触手
